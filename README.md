# NBT library

This is a wip for a high level nbt lib

## Building:
1) Make folder `build`
2) Run `cmake ..` in folder build
3) Run `make`
4) You should have mapart.exe

You will need cmake, make and a c++ compiler

## Nbt Tags

1 = Byte

2 = Short

3 = Int

4 = Long

5 = Float

6 = Double

7 = Byte Array

8 = String

9 = List

10 = Compound

11 = Int array

12 = long array
